# pokemon-resale-dashboard
An interactive financial dashboard tracking 4 months of sealed Pokémon product resale (Jun–Sep 2025) across eBay, TCGPlayer, and Facebook Marketplace.

What's tracked: Revenue, profit, ROI, and margin broken down by month, product category, and platform across 211 transactions.
Built with: Microsoft Excel for data collection and cleaning · Vanilla HTML, CSS, and JavaScript for the dashboard · Canvas API for all charts — no libraries
